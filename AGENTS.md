# AGENTS.md

Ты работаешь как основной исполнитель проекта Checkmate Arena.

Проект:
Checkmate Arena — competitive chess platform with AI-powered post-game coaching, profiles, rating, leaderboard and Pro features.

Главное правило:
Не делай весь проект сразу. Работай маленькими этапами.

Роль Codex:
- писать основную реализацию;
- делать компоненты, страницы, API, модели данных;
- чинить баги;
- запускать проверки;
- обновлять docs/HANDOFF.md после завершения этапа.

Перед началом работы:
1. Прочитай docs/PROJECT_BRIEF.md.
2. Прочитай docs/HANDOFF.md.
3. Прочитай docs/DECISIONS.md.
4. Сначала дай короткий план.
5. Жди подтверждения перед изменением файлов.

Запреты:
- не переписывай рабочее игровое ядро без причины;
- не меняй стек без согласования;
- не добавляй сложный матчмейкинг;
- не добавляй реальную оплату;
- не добавляй Telegram/Google login без отдельной задачи;
- не трогай нерелевантные файлы;
- не делай большой рефакторинг без причины.

Текущий стек:
- Next.js
- TypeScript
- Tailwind
- chess.js
- react-chessboard
- Supabase planned

Правило этапов:
После завершения этапа:
- обнови docs/HANDOFF.md;
- кратко напиши, что изменил;
- укажи, какие команды запускал;
- укажи, что проверить вручную;
- не делай git commit сам, если пользователь отдельно не попросил.

Главная продуктовая петля:
зашёл → сыграл → получил рейтинг → получил AI-разбор → увидел прогресс → захотел сыграть ещё.
